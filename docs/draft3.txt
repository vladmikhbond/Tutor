@2 Логічний тип (bool)

@3 Це числовий тип з лише двома значеннями: True і False.

@3 Складені умови складаються з простих умов за допомогою логічних операцій:  { and, or, not }.

@3 Вираз { 0 < x < 1 } означає те саме, що { 0 < x and x < 1 }.

@3 Нулі і порожні рядки Пітон сприймає як False, все інше як True.

@2 Порядок обчислення логічних виразів

@3 Логічні вирази обчислюються за короткою схемою.
Позначки:
T - вираз, який трактується як True, 
F - вираз, який трактується як False
X - довільний вираз
{{
T or X = T
F or X = X

T and X = X
F and X = F
}}
