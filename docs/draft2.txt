@1 1.02 Типи даних

@2 Числа

@3 Числа бувають 
- цілі (int):  2, 5, 999999999999999999999999999
- дійсні (float): 3.14, 3. , .14, 1.23e45, 1.23E-45
- комплексні (complex): 4 + 5.3j
 
@2 Рядки

@3 Рядкові літерали укладаються в лапки - одинарні або подвійні.

@3 Багаторядкові літерали укладаються в трикратні лапки.

@3 Два літерали, що стоять поруч, інтерпретатор сприймає як один.

@3 Символи всередині рядків можуть бути екрановані:  \t, \n \u1234

@3 У Python 3 для кодування символів за замовчуванням використовується UTF-8.
У Python 2 для кодування символів за замовчуванням використовується ASCII.

@2 Об'єкти

@3 Усі значення в Пітоні є об'єктами.

@3 Тип будь-якого значення повертає вбудована функція type().
{{
>>> type(1)
<class 'int'>
}}

@3 Кожен об'єкт, розташований в пам'яті, отримує унікальний ідентифікатор.
{{
>>> id(555)
1758751865712
}}

@2 Імена та вирази

@3 У Пітоні імена складаються з букв, цифр і підкреслень.

@3 Арифметичні операції: 
{{
+   -   *   **   /   %   //     
}}

@3 Операції з рядками:  +   *

@2 Присвоювання

@3 Операція присвоювання полягає в тому, що якомусь об'єкту надають (присвоюють) ім'я.

@3 {ім'я = вираз} - обчислюється значення виразу, ім'я пов'язується з обчисленим значенням.
{{
x = 2 + 3  
}}

@3 {ім'я = ім'я2} - ім'я зв'язується зі значенням імені2 (тепер в одного об'єкта два імені).
{{
y = x  
}}

@3 З мови С запозичені комбіновані присвоювання.
{{
+=, -=, тощо.
}}

@3 "Відсвоювання".
{{
del x
del y     
}}

@3 У Пітоні можна робити ланцюжкові присвоювання, хоча присвоювання не вважається операцією.
{{
x = y = 1   # OK
x = (y = 1) # error
}}

@2 Коментарі

@3 Однорядкові коментарі починаються із символу #.

@3 Багаторядкові коментарі це строкові літерали в потрійних лапках.


@2 Вихідний код

@3 Вихідний код програми за замовчуванням зберігається у кодуванні UTF-8.

@3 Якщо рядок завершується символом '\' , то наступний рядок вважається його продовженням.

@3 Коли в рядку немає балансу дужок, символ продовження можна не писати.

@3 Особливістю Пітона є двовимірний синтаксис.
