Задачники (Workbooks)

Назва задачника має бути унікальною. Після створення задачника її не можна змінювати.

Задачі додаються і видаляються з задачника кліком миші по списку задач внизу сторінки. Список задач може бути відфільтроваий (поле фільтру знаходиться у правому верхньому куті сторінки).

Коли задачник відкритий, його задачі доступні для вирішення. Момент відкриття задачника - в полі Open Time. Задачник залишається відкритии на протязі певної кількості хвилин. Ця кількість - в полі Open Minutes.

Задачник доступний для користувачів, чії імена віповідають зразку в полі User Sample.
